# Victor Yu - Personal Webpage

This is the personal webpage for Victor Yu, an EE undergraduate and quantum science researcher at UCLA.
This site was fully generated by Windsurf in an effort to learn about vibe coding.

## Structure

- **index.html**: Homepage with hero section and light/dark mode toggle
- **pages/**: Research, Hobbies, and Life Lessons pages
- **assets/css/styles.css**: Main styles with themes
- **assets/js/main.js**: Theme toggle and interactive scripts
- **data/lessons.json**: Placeholder JSON for Life Lessons

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/victortcyu/personal-website.git
   cd personal-website
   ```
2. Serve locally (optional):
   ```bash
   python -m http.server 8000
   ```
3. Open `http://localhost:8000` in your browser.

## Updating Content

- Edit `data/lessons.json` for Life Lessons.
- Edit `data/placeholders.json` for bio and page content.
- Replace placeholder images in `assets/images/`.
- Modify HTML in `index.html` or files under `pages/`.
- Run the content validation script:
  ```bash
  python scripts/generate-content.py
  ```
  This validates your JSON content files before deployment.

## Deployment

This site is hosted via GitHub Pages. Updates to the `main` branch are automatically deployed.
